# Moclobemide

moclobemide

## Basic Information

**Summary**

A reversible MAOI which is prescribed to treat depression and anxiety. Its reversibility and short-acting length of effects often means it is used in "pharmahuasca" preparations.

**Dose**

| Oral   |        |
| ------ | ------ |
| Light  | 150mg  |
| Common | 300mg  |
| Strong | 600mg. |

**Duration**

| All ROAs |               |
| -------- | ------------- |
| Onset    | 30-90 minutes |
| Duration | 2-6 hours     |
